Markov Decision Process presentation (reveal.js)

Open locally
------------
1. Unzip this folder anywhere.
2. Open mdp.html in a browser (Chrome or Firefox recommended).

   file:// works, but some browsers restrict local files. If math or
   speaker notes fail, use a tiny local server instead:

     python3 -m http.server 8000

   Then visit http://localhost:8000/mdp.html

Internet
--------
KaTeX (math formulas) loads from cdn.jsdelivr.net on first view. You need
network access for equations to render; slides and navigation work offline.

Controls
--------
Arrow keys / click: next slide
S: speaker notes (popup)
F: fullscreen
Esc: overview
